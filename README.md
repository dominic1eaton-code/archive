# I. ONTOLOGICAL PRIMITIVES

## Definition 1 — Monad

A **Monad** is the minimal unit of state that preserves identity under bounded change.

Formally:

A Monad m is a tuple

m := (s, ε, τ)

where:

* s ∈ S is an internal state
* ε > 0 is the resolution bound (minimum distinguishable change)
* τ is an internal persistence timescale

Invariant:
|Δs| < ε  ⇒  m is the *same* Monad

Interpretation:

* Physics: localized state
* Psychology: emotion, belief, intention
* Cognition: atomic mental content

Monads are **not static**; they are *stable under bounded variation*.

---

## Definition 2 — Monad Space

Let M be the set of all Monads.

M is not a vector space, but a **topological state space** with metric:

d(m_i, m_j) := distinguishability(m_i, m_j)

where:
d < ε ⇒ indistinguishable
d ≥ ε ⇒ distinct Monads

# II. INTERACTION STRUCTURES

## Definition 3 — Dualon

A **Dualon** is a structured interaction between two (or more) Monads that produces emergent behavior not reducible to either alone.

Formally:

A Dualon D is a morphism:

D : m_i × m_j → m_k

subject to:
m_k ≠ m_i
m_k ≠ m_j

and energy / information exchange:

ΔE_ij ≠ 0
ΔI_ij ≠ 0

Interpretation:

* Physics: interaction, force, coupling
* Psychology: decision-making, conflict, synthesis
* Cognition: comparison, inference, choice

A Dualon is **processual**, not an object.

## Definition 4 — Trilon / Polyon

A **Polyon System** is a higher-order interaction network:

P := (M_P, D_P, G)

where:

* M_P ⊆ M is a finite set of Monads

For |M_P| = 3, the system is a **Trilon**
