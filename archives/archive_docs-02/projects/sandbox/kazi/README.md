C2C C2B B2C digital labor management platform
digital labor management platform



## application modules, core features
* space (digital workspace, digital profile)
    * portfolio; personal projects portfolio management
        - solutions,products,services,projects (development) management
    * docs; personal document management
    * calender; personal schedule management
    * email; personal communications management
* finance
* marketplace (digital laborer,contractor,gig,freelancer marketplace)
* community (digital labor social network)
* work (digital work/task management system)
* search; info, help, SEO
* events, calender


* user account
    * user profile,dashboard,portal
    * digital workspace
    * digital marketplace profile
    * digital work profile
    * digital SEO management
    * digital community profile
    * digital finance account
    * digital personal communications space, social media, email, chat
* group,team,organization account




* marketplace
    * services marketplace
        * contractor marketplace
            * request for porposal project multicast, tagged proposal
                * subscribers to RFP tagged project recieve notification,
                * make easier to find contractors to complete a desired project; make it easier to find projects contracts a contractor would like to work
        * on demand gig worker, task worker marketplace
            * e.g. find delivery service, data entry clerk, call center associcate, short term easy to onboard repeatable trackable task workers marketplace
        * volunteer, start up worker marketplace
            * find startup workers interested in equity based compensation
            * find volunteers for NPO based events and activities
        * consultant marketplace
        * learning,education,courses marketplace
    * products marketplace
        * software products marketplace; "software app store", capterra
            - one stop shop for easily finding any software for any user need/desire
            * business softwares
            * gaming softwares
            * work management, personal organization software
            * communication software
            * misc. software
            - tile based UI software list
                - e.g. search: marketing software
                    - results: mailchimp tile
                        - link to mailchimp site
                        - ratings, reviews
                        - link to youtube how to videos, social media
                        - company/software information
