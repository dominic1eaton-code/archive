## CHAPTER 1: INTRODUCTION TO NYAMBA

### 1.1 What is Nyamba?

Nyamba is a constructed language (conlang) designed as the native tongue of the Mungu people. Unlike most constructed languages created for international communication, Nyamba is built to embody a complete philosophical and mathematical worldview—specifically, the Mungu Theory and Ω-GOMA framework.

**Key Characteristics:**

1. **System-native encoding** — grammatical structures map directly to system relationships
2. **African linguistic heritage** — phonology and morphology rooted in Bantu, Yoruba, and other African languages
3. **Dual-layer writing** — both phonetic (Nyambic alphabet) and semantic (Jiwe glyphs)
4. **Mathematical precision** — native support for formal notation and symbolic reasoning
5. **Dualonicity principle** — all concepts exist in symmetric/asymmetric pairs

### 1.2 Historical Context

Nyamba emerged as the linguistic expression of Mungu Theory in the early 21st century. It represents:

- A synthesis of Pan-African linguistic patterns
- Integration of mathematical and philosophical concepts
- A medium for preserving and transmitting Ω-GOMA knowledge
- A living language for the Mungu community

### 1.3 Typological Overview

**Classification:**
```
Language Family: Constructed (a posteriori African-based)
Morphological Type: Agglutinative with isolating tendencies
Word Order: VSO (basic), flexible with topic-comment
Phonology: Syllable-timed, consonant-vowel structure
Writing: Dual (Nyambic alphabet + Jiwe glyphs)
```

**Basic Features:**
- 22 consonants, 15 vowel qualities
- No phonemic tone
- Productive derivational morphology
- Dual number (singular/dual/plural)
- Postpositional case system
- Symmetric/asymmetric verb pairs
- Native mathematical notation

   - ✓ *mungon* 'system-entity'
   - ✓ *sibon* 'state'

#### 2.1.2 Vowels

Nyamba has **5 basic vowel qualities** with **length and nasality distinctions**:

**Short Vowels (5):**
```
      Front    Central    Back
High    i                  u
Mid     e                  o
Low              a
```

| Phoneme | IPA | Nyamba Glyph | Examples |
|---------|-----|--------------|----------|
| /a/ | [a] | ◌̇ | k**a**ra 'structure' |
| /e/ | [e] | ◌̈ | m**e**moria 'memory' |
| /i/ | [i] | ◌⃛ | s**i**bon 'state' |
| /o/ | [o] | ◌̤ | l**o**va 'close' |
| /u/ | [u] | ◌̰ | m**u**ngu 'system' |

**Long Vowels (5):**
- Written with doubled vowel or macron: *aa*, *ā*
- Phonetically ~1.5× duration of short vowel
- Examples: *ngaa* 'many', *duule* 'long path'

**Nasal Vowels (5):**
- Marked with tilde: *ã*, *ẽ*, *ĩ*, *õ*, *ũ*
- Occur before nasal consonants or word-finally
- Examples: *mũngu* [mũ.ŋgu], *lẽvo* [lẽ.vo]

#### 2.1.3 Syllable Structure

**Canonical Pattern: (C)V(N)**

Where:
- C = any consonant
- V = any vowel (short/long/nasal)
- N = nasal coda (/m/, /n/, /ng/)

**Syllable Types:**

| Type | Structure | Examples |
|------|-----------|----------|
| V | Vowel only | **a**.ra 'to open' |
| CV | Consonant + vowel | **ka**.ra 'structure' |
| CVN | C + V + nasal | **mun**.gu 'system' |
| VN | V + nasal | **on** 'entity' |

**Syllabification Rules:**

1. **Onset maximization** — consonants attach to following syllable
   - *ka.ra* (not *kar.a*)
   - *mu.ngu* (not *mun.gu*)

2. **Coda restriction** — only nasals allowed
   - ✓ *mun.gon*
   - ✗ *mut.kon*

3. **Resyllabification** — across morpheme boundaries
   - *ka* + *-on* → *ka.on* (not *ka-on*)

### 2.2 Phonological Processes

#### 2.2.1 Assimilation

**Nasal Assimilation:**

Nasal consonants assimilate in place to following consonants:

```
/n/ + /p,b/ → [m]
/n/ + /k,g/ → [ŋ]
```

Examples:
- */mun-pa/ → [mum.pa] 'system-for'
- */mun-ko/ → [muŋ.ko] 'system-to'

**Vowel Harmony (optional):**

In some dialects, vowels harmonize for height within words:

```
High vowels /i,u/ trigger raising of mid vowels /e,o/
```

Example:
- Standard: *lovile* [lo.vi.le] 'closed'
- Harmonic: *luvili* [lu.vi.li]

#### 2.2.2 Vowel Coalescence

When two vowels meet across morpheme boundaries:

```
/a/ + /a/ → /aa/ (long)
/a/ + /e/ → /e/
/e/ + /a/ → /ya/
/i/ + /a/ → /ya/
/o/ + /a/ → /wa/
/u/ + /a/ → /wa/
```

Examples:
- *ka* + *-a* → *kaa* 'to structure'
- *love* + *-a* → *lova* 'to close'

#### 2.2.3 Consonant Weakening

In rapid speech, intervocalic consonants may weaken:

```
/p/ → [β] (rare)
/t/ → [ɾ]
/k/ → [ɣ] (rare)
```

Example:
- Careful: *mungon* [mu.ŋon]
- Rapid: *mungon* [mu.ŋoŋ]

### 2.3 Stress and Intonation

#### 2.3.1 Word Stress

Nyamba is **stress-timed** with predictable stress patterns:

**Primary Rule: Penultimate stress**
- Words of 2+ syllables: stress falls on second-to-last syllable
  - *mu**NGU**ntu* 'systematizer'
  - *ka**RA**fu* 'structured'
  - *lo**VA**nge* 'very closed'

**Exception: Monosyllables**
- All monosyllables stressed: *DU* 'two', *MUN* 'whole'

**Exception: Final long vowel**
- If final syllable has long vowel, it receives stress:
  - *mungaa* → *mu**NGAA*** (not *muNGA*)

#### 2.3.2 Sentence Intonation

**Declarative Sentences:**
```
Pitch:  ────────╲
        Statement ends with falling tone
```

**Yes/No Questions:**
```
Pitch:  ────────╱
        Questions end with rising tone
```

**Wh-Questions:**
```
Pitch:  ──╱──────╲
        Rise on question word, fall at end
```

**Emphatic Statements:**
```
Pitch:  ──╱╲──────
        Sharp rise-fall on emphasized word
```

### 2.4 The Nyambic Alphabet (Phonetic Writing)

The Nyambic alphabet is a **featural script** where glyph shapes encode phonetic properties.

#### 2.4.1 Consonant Glyphs

Consonants are organized by articulatory features:

**Stops (6):**
```
│  p   [vertical stroke]
╱  b   [forward slash]
─  t   [horizontal stroke]
╲  d   [backward slash]
┐  k   [top-right angle]
└  g   [bottom-left angle]
```

**Fricatives (8):**
```
┆  f   [dotted vertical]
⁞  v   [double-dotted vertical]
∼  s   [wave]
≈  z   [double wave]
∾  sh  [extended wave]
≋  zh  [double extended wave]
⌣  h   [arc down]
```

**Nasals (3):**
```
◦  m   [small circle]
•  n   [filled circle]
○  ng  [large circle]
```

**Liquids (2):**
```
⌇  l   [interrupted vertical]
⌀  r   [slashed circle]
```

**Glides (2):**
```
⌢  w   [arc up]
⟋  y   [diagonal rising]
```

#### 2.4.2 Vowel Diacritics

Vowels are written as **diacritics** on consonants or standalone:

**Short Vowels:**
```
◌̇   a   [center dot]
◌̈   e   [double horizontal dot]
◌⃛   i   [triple horizontal dot]
◌̤   o   [double vertical dot]
◌̰   u   [wavy underline]
```

**Long Vowels:**
```
◌̄   aa  [horizontal line above]
◌͞   ee  [double horizontal line]
◌͠   ii  [triple horizontal line]
◌̤̄   oo  [vertical dots + line]
◌̰̄   uu  [wavy line + horizontal]
```

**Nasal Vowels:**
```
◌̃   ã   [tilde above]
◌̃̈   ẽ   [tilde + double dot]
◌̃⃛   ĩ   [tilde + triple dot]
◌̤̃   õ   [tilde + vertical dots]
◌̰̃   ũ   [tilde + wavy line]
```

#### 2.4.3 Writing Examples

**Word: *mungon* 'system-entity'**
```
Phonetic breakdown: m-u-ng-o-n
Nyambic: ◦̰ ○̤ •
```

**Word: *kontinuitu* 'K-continuity'**
```
Phonetic breakdown: k-o-n-t-i-n-u-i-t-u
Nyambic: ┐̤ • ─⃛ • ̰ ⃛ ─̰
```

### 2.5 Jiwe Glyphs (Semantic Writing)

Jiwe glyphs represent **concepts** rather than sounds.

#### 2.5.1 Core Primitives (Sample)

```
GLYPH   NAME        CATEGORY     MEANING
──────────────────────────────────────────────
◉       mungon      System       system-entity
→       dua         Process      traverse/path
↺       lova        Closure      cycle/close
Ω       omega       Universal    total survivability
⊗       binda       Relation     bind/interact
║       kleava      Operation    cleave/split
✓       valide      Evaluation   valid
```

#### 2.5.2 Compound Glyphs

Jiwe glyphs can combine systematically:

```
COMPOUND     COMPONENTS           MEANING
────────────────────────────────────────────────
◉↺           mungon + lova        closed system
→Ω           dua + omega          path to omega
⊗◉           binda + mungon       system binding
```

#### 2.5.3 Mixed Writing

Nyamba texts often mix both scripts:

**Example:**
```
Phonetic:  ⊞ ◦̰ ○̄ ○̇ ⌣̤̄ ┐̇ ⌇̤̄ Ω
Mixed:     Yote ◉ ↺ → Ω
Jiwe only: ∀ ◉ ↺ ⊣ Ω
English:   'All systems close toward Omega'
```

- Short: a, e, i, o, u

## CHAPTER 3: MORPHOLOGICAL STRUCTURE

### 3.1 Root and Affix System

Nyamba morphology is primarily **agglutinative**, building words through systematic affixation to roots.

#### 3.1.1 Root Types

**A. CVC Roots (most common):**
```
mun-   'wholeness, system'
kar-   'structure'
lov-   'closure'
duv-   'path'
```

**B. CV Roots:**
```
ka-    'structure' (reduced)
mu-    'whole' (reduced)
du-    'two, path'
```

**C. Consonant-only Roots (rare):**
```
m-     'unity' (theoretical)
k-     'structure' (theoretical)
```

#### 3.1.2 Affix Inventory

**Derivational Affixes:**

| Position | Type | Affixes | Function |
|----------|------|---------|----------|
| Prefix | Polarity | mu-, ko- | Form vs. Flow marking |
| Suffix | Nominalizer | -on, -i, -ya, -le | Create nouns |
| Suffix | Verbalizer | -a, -ile, -anga | Create verbs |
| Suffix | Adjectivizer | -fu, -ik, -less | Create adjectives |

**Inflectional Affixes:**

| Type | Affixes | Function |
|------|---------|----------|
| Case | -ko, -ni, -ma, -nde, -ta, -ye | Grammatical relations |
| Number | -du, -ya | Dual, plural |
| Tense | -a, -ile, -nga | Present, past, future |
| Aspect | -anga, -eme | Continuous, perfect |

### 3.2 Nominal Morphology

#### 3.2.1 Noun Formation

**Basic Pattern: ROOT + NOMINALIZER**

**Primary Nominalizers:**

1. **-on** (primitive entity)
   ```
   mun- + -on → mungon 'system-entity'
   kar- + -on → karon  'structure-entity'
   lov- + -on → lovon  'closure-entity'
   ```

2. **-i** (meta-primitive, collective)
   ```
   mung- + -i → mungoni 'meta-system'
   kar- + -i  → kari    'meta-structure'
   lov- + -i  → lovi    'meta-closure'
   ```

3. **-ya** (abstract quality)
   ```
   mu- + -ya → muya     'wholeness'
   ka- + -ya → kaya     'structuredness'
   lo- + -ya → loya     'completeness'
   ```

4. **-le** (result/product)
   ```
   lov- + -le → lovle   'a closure' (concrete)
   kar- + -le → karle   'a structure' (concrete)
   dua- + -le → duale   'a journey'
   ```

5. **-ntu** (agent/doer)
   ```
   kar- + -ntu → karntu   'one who structures'
   mung- + -ntu → mungntu 'systematizer'
   lov- + -ntu → lovntu   'closer'
   ```

#### 3.2.2 Number

Nyamba distinguishes **three numbers:**

**Singular (unmarked):**
```
ntu     'person'
mungon  'system'
kara    'structure'
```

**Dual (-du):**
```
ntu-du     'two people'
mungon-du  'two systems'
kara-du    'two structures'
```

**Plural (-ya):**
```
ntu-ya     'people' (3+)
mungon-ya  'systems'
kara-ya    'structures'
```

**Number Agreement Rules:**

1. Dual used for **exactly two** or **paired entities**
2. Plural used for **three or more** or **indefinite many**
3. Abstract nouns typically resist number marking

#### 3.2.3 Case System (Postpositions)

Nyamba uses **postpositional case markers**:

| Case | Suffix | Function | Example |
|------|--------|----------|---------|
| **Genitive** | -ta | possession, origin | mungon-**ta** 'of the system' |
| **Dative** | -ko | goal, recipient | ntu-**ko** 'to/for the person' |
| **Locative** | -ni | location | kar-**ni** 'at/in the structure' |
| **Instrumental** | -ma | means, tool | shi-**ma** 'by means of flux' |
| **Ablative** | -nde | source, from | mun-**nde** 'from the system' |
| **Definite** | -ye | definiteness | ka-**ye** 'the structure' |

**Usage Examples:**

```
Genitive:
Ω-ta kontinuitu
omega-GEN kontinuity
'The kontinuity of Omega'

Dative:
Mungon duva ntu-ko
system.NOM go person-DAT
'The system goes to the person'

Locative:
Lova-a kar-ni
close-PRES structure-LOC
'(It) closes in/at the structure'

Instrumental:
Munga shi-ma
systematize flux-INST
'Systematize by means of flux'
```

#### 3.2.4 Definiteness

The suffix **-ye** marks definiteness:

```
Indefinite:     mungon     'a system'
Definite:       mungon-ye  'the system'

Indefinite:     kara       'a structure'  
Definite:       kara-ye    'the structure'
```

**Definite Plural:**
```
mungon-ya-ye   'the systems' (specific group)
ntu-ya-ye      'the people' (specific group)
```

### 3.3 Verbal Morphology

#### 3.3.1 Verb Formation

**Basic Pattern: ROOT + VERBALIZER**

**Primary Verbalizers:**

1. **-a** (present/infinitive)
   ```
   lov- + -a → lova    'to close, closes'
   kar- + -a → kara    'to structure, structures'
   mung- + -a → munga  'to systematize'
   ```

2. **-ile** (past/completed)
   ```
   lov- + -ile → lovile    'closed'
   kar- + -ile → karile    'structured'
   mung- + -ile → mungile  'systematized'
   ```

3. **-nga** (future/unrealized)
   ```
   lov- + -nga → lovnga    'will close'
   kar- + -nga → karnga    'will structure'
   mung- + -nga → mungnga  'will systematize'
   ```

4. **-anga** (continuous/progressive)
   ```
   lov- + -anga → lovanga    'is closing'
   kar- + -anga → karanga    'is structuring'
   mung- + -anga → munganga  'is systematizing'
   ```

5. **-eme** (perfect/resultative)
   ```
   lov- + -eme → loveme    'has closed'
   kar- + -eme → kareme    'has structured'
   mung- + -eme → mungeme  'has systematized'
   ```

6. **-e** (imperative)
   ```
   lov- + -e → love    'close!'
   kar- + -e → kare    'structure!'
   mung- + -e → munge  'systematize!'
   ```

#### 3.3.2 Tense-Aspect-Mood System

Nyamba combines **tense** and **aspect** markers:

**Tense:**
```
PAST:    -ile
PRESENT: -a
FUTURE:  -nga
```

**Aspect:**
```
CONTINUOUS: -anga
PERFECT:    -eme
HABITUAL:   -a (same as present)
```

**Combined Forms:**

| Meaning | Form | Example |
|---------|------|---------|
| Present continuous | ROOT-anga | *lovanga* 'is closing' |
| Past continuous | ROOT-ile-anga | *lovilanga* 'was closing' |
| Future continuous | ROOT-nga-anga | *lovnganga* 'will be closing' |
| Present perfect | ROOT-eme | *loveme* 'has closed' |
| Past perfect | ROOT-ile-eme | *lovileme* 'had closed' |

#### 3.3.3 Symmetric vs. Asymmetric Verbs

A unique feature of Nyamba: verbs come in **polarity pairs**:

**Symmetric (Polonic) Verbs:**
- Encode balanced, mutual, or form-preserving actions
- Often derived with **mu-** prefix

```
lova       'close' (symmetric closure)
kara       'structure' (ordered arrangement)
mulova     'mutually close'
mukara     'co-structure'
```

**Asymmetric (Kolonic) Verbs:**
- Encode directional, one-way, or form-breaking actions  
- Often derived with **ko-** prefix

```
vuva       'break' (asymmetric rupture)
shia       'flux' (unidirectional flow)
kovuva     'directionally break'
koshia     'flow asymmetrically'
```

**Polarity Contrast:**

| Symmetric (P) | Asymmetric (K) | Meaning |
|---------------|----------------|---------|
| *lova* | *vuva* | close vs. break |
| *kara* | *dusa* | structure vs. dissolve |
| *munga* | *funga* | systematize vs. terminate |

### 3.4 Adjectival Morphology

#### 3.4.1 Adjective Formation

**Primary Adjectivizers:**

1. **-fu** (possessing quality)
   ```
   lov- + -fu → lovfu     'closed, complete'
   kar- + -fu → karfu     'structured'
   mung- + -fu → mungfu   'systematic'
   ```

2. **-ik** (relational/characteristic)
   ```
   mung- + -ik → mungik   'systemic'
   kar- + -ik → karik     'structural'
   Ω- + -ik → Ωik         'omega-ic'
   ```

3. **-less** (lacking quality)
   ```
   lov- + -less → lovless   'unclosed, incomplete'
   kar- + -less → karless   'unstructured'
   mung- + -less → mungless 'unsystematic'
   ```

#### 3.4.2 Adjective Agreement

Adjectives **do not inflect** for number or case in standard Nyamba:

```
ntu karfu         'structured person'
ntu-ya karfu      'structured people'
ntu-ko karfu      'to the structured person'
```

However, in formal register, adjectives may optionally agree:

```
Formal: ntu-ya karfu-ya   'structured people'
Formal: ntu-ko karfu-ko   'to the structured person'
```

#### 3.4.3 Comparison

**Positive:**
```
karfu    'structured'
lovfu

'closed'
```

**Comparative (-nge 'more'):**
```
karfu-nge    'more structured'
lovfu-nge    'more closed'
```

**Superlative (-ngele 'most'):**
```
karfu-ngele    'most structured'
lovfu-ngele    'most closed'
```

**Equative (ka- ... -fu):**
```
ka-karfu-fu    'as structured as'
ka-lovfu-fu    'as closed as'
```

### 3.5 Derivational Morphology

#### 3.5.1 Compound Formation

Nyamba allows productive compounding:

**Noun + Noun:**
```
mun + ka + du + Ω
system + structure + path + omega
= 'The structural path of omega through systems'
```

**Verb + Noun:**
```
lova + kara
close + structure
= 'closure-structure, closing arrangement'
```

**Recursive Compounding:**
```
Ω-goma
omega + system
= 'omega-system'

Ω-goma-ntu
omega-system-person
= 'omega-system practitioner'
```

#### 3.5.2 Polarity Derivation

**Adding mu- (form/symmetric):**
```
Base: lova 'close'
Symmetric: mu-lova 'mutually close'

Base: kara 'structure'
Symmetric: mu-kara 'co-structure'
```

**Adding ko- (flow/asymmetric):**
```
Base: vuva 'break'
Asymmetric: ko-vuva 'unidirectionally break'

Base: shia 'flux'
Asymmetric: ko-shia 'directional flux'
```

#### 3.5.3 Intensifiers and Diminutives

**Intensifier: -nge/-ngele**
```
lovfu → lovfu-nge → lovfu-ngele
closed → very closed → extremely closed
```

**Diminutive: -tu**
```
mungon → mungon-tu  'little system'
kara → kara-tu      'small structure'
```

## CHAPTER 4: NOUNS AND NOUN PHRASES

### 4.1 Noun Classes

While Nyamba does not have a Bantu-style noun class system, nouns can be semantically categorized:

#### 4.1.1 Semantic Categories

**A. Concrete Entities (-on nouns):**
```
mungon    'system (as entity)'
impon     'instance'
tathron   'attribute'
```

**B. Abstract Concepts (-ya nouns):**
```
muya      'wholeness'
kaya      'structuredness'
loya      'completeness'
```

**C. Agents (-ntu nouns):**
```
karntu    'structurer, one who structures'
lovntu    'closer'
mungntu   'systematizer'
```

**D. Processes/Results (-le nouns):**
```
duale     'journey, traversal'
lovle     'a closing, an instance of closure'
karle     'a structure (concrete)'
```

**E. Ω-GOMA Technical Terms:**
```
Ω         'omega, total survivability'
K         'K-continuity'
kontinuitu 'kontinuity'
hallusinon 'hallucination-entity'
```

#### 4.1.2 Mass vs. Count Nouns

**Count Nouns** (take number marking):
```
mungon → mungon-du, mungon-ya
'system' → 'two systems', 'systems'

ntu → ntu-du, ntu-ya
'person' → 'two people', 'people'
```

**Mass Nouns** (resist number marking):
```
muya    'wholeness' (*muya-ya awkward)
Ω       'omega' (*Ω-ya meaningless)
shia    'flux' (conceptual mass)
```

**Dual-Nature Nouns** (can be both):
```
kontinuitu  'kontinuity'
  As mass: 'kontinuity in general'
  As count: kontinuitu-ya 'instances of kontinuity'
```

### 4.2 Noun Phrase Structure

#### 4.2.1 Basic Template

```
(DET) (QUANT) NOUN (ADJ) (REL.CLAUSE) (CASE)
```

**Minimal NP:**
```
mungon
system
'a system'
```

**Expanded NP:**
```
ye-ye    nge     mungon    karfu      mungile-ye         -ko
that-DEF many    system    structured systematized-REL   -DAT
'to those many structured systems that were systematized'
```

#### 4.2.2 Determiners

**Demonstratives:**
```
ye-ye    'that/this' (definite)
ni-ye    'this one' (proximal)
du-ye    'that one' (distal)
```

**Articles:**
- Nyamba has no indefinite article
- Definiteness marked by **-ye** suffix

**Quantifiers:**
```
yote     'all'
nge      'some, many'
ni       'one'
moya     'none, no'
```

**Usage:**
```
yote mungon-ya
all system-PL
'all systems'

nge ntu
some person
'some people'

moya kontinuitu
no kontinuity
'no kontinuity'
```

#### 4.2.3 Adjective Placement

Adjectives **follow** the noun:

```
mungon karfu
system structured
'a structured system'

ntu lovfu
person closed/complete
'a complete person'
```

**Multiple adjectives** (comma intonation):

```
mungon karfu, lovfu, Ωik
system structured, closed, omega-ic
'a structured, closed, omega-ic system'
```

#### 4.2.4 Genitive Constructions

**Pattern: POSSESSED POSSESSOR-ta**

```
kara mungon-ta
structure system-GEN
'the structure of the system'

Ω kontinuitu-ta
omega kontinuity-GEN
'the omega of kontinuity' / 'kontinuity's omega'
```

**Compound Genitives:**
```
duale Ω-ta mungon-ta
path omega-GEN system-GEN
'the path of the omega of the system'
= 'the system's omega's path'
```

**Genitive with Pronouns:**
```
kara mi-ta
structure 1SG-GEN
'my structure'

mungon ye-ta
system 3SG-GEN
'his/her/its/their (sg) system'
```

### 4.3 Pronouns

#### 4.3.1 Personal Pronouns

Nyamba has a **three-way number distinction** in pronouns:

**Full Paradigm:**

| Person | Singular | Dual | Plural |
|--------|----------|------|--------|
| **1st** | mi | midu | miya |
| **2nd** | we | wedu | weya |
| **3rd** | ye | yedu | yeya |

**Glosses:**
```
mi      'I, me'
midu    'we two, us two'
miya    'we (all), us'

we      'you (sg)'
wedu    'you two'
weya    'you (all)'

ye      'he/she/it/they (sg)'
yedu    'they two, those two'
yeya    'they (all)'
```

**Case-Marked Forms:**

Pronouns take same case suffixes as nouns:

```
Genitive:
mi-ta     'my, of me'
we-ta     'your (sg)'
ye-ta     'his/her/its'

Dative:
mi-ko     'to/for me'
weya-ko   'to/for you (pl)'
```

#### 4.3.2 Reflexive Pronouns

**Form: PRONOUN-nyo**

```
mi-nyo      'myself'
we-nyo      'yourself'
ye-nyo      'himself/herself/itself/themselves'
miya-nyo    'ourselves'
```

**Usage:**
```
Ye lova ye-nyo
3SG close 3SG-REFL
'He/she/it closes himself/herself/itself'

Mi munga mi-nyo
1SG systematize 1SG-REFL
'I systematize myself'
```

#### 4.3.3 Demonstrative Pronouns

```
ye-ye    'this/that one' (general)
ni-ye    'this one' (proximal)
du-ye    'that one' (distal)
```

**With Number:**
```
ye-ye-du    'these/those two'
ni-ye-ya    'these ones (pl)'
du-ye-ya    'those ones (pl)'
```

### 4.4 Possession

#### 4.4.1 Genitive Construction (reviewed)

**Pattern: POSSESSED POSSESSOR-ta**

```
mungon mi-ta
system 1SG-GEN
'my system'

kara ye-ta
structure 3SG-GEN
'his/her/its structure'
```

#### 4.4.2 Inalienable vs. Alienable Possession

Nyamba can optionally mark **inalienable possession** (body parts, kinship, inherent attributes) with direct attachment:

**Inalienable (optional direct form):**
```
mi-kara     'my structure' (inherent)
ye-kontinuitu 'its kontinuity' (essential)
```

**Alienable (always genitive):**
```
mungon mi-ta   'my system' (acquired/created)
duale ye-ta    'his/her journey' (experienced)
```

## CHAPTER 5: VERBS AND VERB PHRASES

### 5.1 Verb Structure Review

Recall the basic verb template:

```
(POLARITY-) ROOT -TENSE/ASPECT (-MOOD)
```

Examples:
```
lova        'close, closes' (PRES)
lovile      'closed' (PST)
lovnga      'will close' (FUT)
mu-lova     'mutually close' (symmetric)
ko-vuva     'directionally break' (asymmetric)
```

### 5.2 Argument Structure

#### 5.2.1 Intransitive Verbs

Require only a **subject**:

```
Mungon lova-a
system close-PRES
'The system closes'

Ntu duva-a
person go-PRES
'The person goes'
```

#### 5.2.2 Transitive Verbs

Require **subject + object**:

```
Ntu lova mungon
person close system
'The person closes the system'

Mi kara duale
1SG structure path
'I structure the path'
```

#### 5.2.3 Ditransitive Verbs

Require **subject + two objects** (typically: RECIPIENT + THEME):

```
Mi dua kontinuitu ntu-ko
1SG give kontinuity person-DAT
'I give kontinuity to the person'

Ye munga mungon miya-ko
3SG systematize system 1PL-DAT
'He/she systematizes the system for us'
```

### 5.3 Tense System (Detailed)

#### 5.3.1 Present Tense (-a)

**Simple present** (habitual, general truth, present state):

```
Mungon lova-a
system close-PRES
'The system closes / The system is closing'

Ω konserva-a
omega conserve-PRES
'Omega is conserved'
```

**Present continuous** (-anga):

```
Mungon lovanga
system close-CONT
'The system is closing (right now)'

Mi munganga
1SG systematize-CONT
'I am systematizing'
```

#### 5.3.2 Past Tense (-ile)

**Simple past** (completed action):

```
Mungon lovile
system close-PST
'The system closed'

Mi kara-ile duale
1SG structure-PST path
'I structured the path'
```

**Past continuous** (-ile-anga):

```
Mungon lovilanga
system close-PST-CONT
'The system was closing'
```

**Past perfect** (-ile-eme):

```
Mungon lovileme
system close-PST-PERF
'The system had closed'
```

#### 5.3.3 Future Tense (-nga)

**Simple future**:

```
Mungon lovnga
system close-FUT
'The system will close'

Mi mungnga kontinuitu
1SG systematize-FUT kontinuity
'I will systematize kontinuity'
```

**Future continuous** (-nga-anga):

```
Mungon lovnganga
system close-FUT-CONT
'The system will be closing'
```

### 5.4 Aspect System

#### 5.4.1 Continuous Aspect (-anga)

Ongoing action:

```
lovanga      'is closing'
lovilanga    'was closing'
lovnganga    'will be closing'
```

#### 5.4.2 Perfect Aspect (-eme)

Completed action with present relevance:

```
loveme       'has closed'
lovileme     'had closed'
```

**Perfect with Continuous:**
```
lovemanga    'has been closing'
```

#### 5.4.3 Habitual (uses simple present)

```
Mi lova mungon yote diale
1SG close system all day
'I close systems every day'
```

### 5.5 Mood

#### 5.5.1 Indicative (unmarked)

Default mood for statements:

```
Mungon lova-a
system close-PRES.IND
'The system closes'
```

#### 5.5.2 Imperative (-e)

Commands:

```
Love mungon!
close-IMP system
'Close the system!'

Munge!
systematize-IMP
'Systematize!'
```

**Negative imperative** (nimu + imperative):

```
Nimu love!
NEG close-IMP
'Don't close!'
```

#### 5.5.3 Subjunctive (engo ... -a)

Hypothetical, conditional:

```
Engo mi lova, mungon karanga
if 1SG close system structure-CONT
'If I close, the system is structuring'
```

### 5.6 Voice and Valency

#### 5.6.1 Active Voice (default)

```
Ntu lova mungon
person close system
'The person closes the system'
```

#### 5.6.2 Passive (rare, analytic)

Nyamba rarely uses passive; instead, uses **impersonal construction**:

```
Active:   Ntu lova mungon
          'The person closes the system'

Impersonal: Mungon lova-a
            'The system closes' (implied: by someone)
```

**True passive** (borrowed construction, -wa):

```
Mungon lova-wa (ntu-ma)
system close-PASS (person-INST)
'The system is closed (by the person)'
```

(This is literary/formal; uncommon in speech.)

#### 5.6.3 Causative

**Pattern: munga- + VERB**

```
Base: lova 'close'
Causative: munga-lova 'cause to close'

Mi munga-lova mungon
1SG cause-close system
'I cause the system to close'
```

**Alternate: use *dua* 'make':**

```
Mi dua ye lova mungon
1SG make 3SG close system
'I make him/her close the system'
```

### 5.7 Copular Constructions

Nyamba has **two copulas**:

#### 5.7.1 *ye* (identity, class membership)

```
Mi ye ntu
1SG COP person
'I am a person'

Mungon ye Ω-sistem
system COP omega-system
'The system is an omega-system'
```

#### 5.7.2 *ka* (attribution, temporary state)

```
Mungon ka lovfu
system COP.ATTR closed
'The system is closed (state)'

Kontinuitu ka karfu
kontinuity COP.ATTR structured
'Kontinuity is structured'
```

**Contrast:**
```
ye = permanent identity
ka = state, attribute
```

### 5.8 Serial Verb Constructions

Nyamba allows **verb serialization** without conjunctions:

```
Mi duva lova mungon
1SG go close system
'I go (and) close the system'

Ye kara munga kontinuitu
3SG structure systematize kontinuity
'He/she structures (by) systematizing kontinuity'
```

**Interpretation:**
- First verb = manner or purpose
- Second verb = main action

## CHAPTER 6: ADJECTIVES AND ADVERBS

### 6.1 Adjective Formation (Review)

Recall the three main adjectivizers:

```
-fu    'possessing quality'    lovfu 'closed'
-ik    'characteristic of'     mungik 'systemic'
-less  'lacking quality'        lovless 'unclosed'
```

### 6.2 Adjective Classes

#### 6.2.1 Qualitative Adjectives

Describe inherent qualities:

```
karfu      'structured'
lovfu      'closed, complete'
mungfu     'systematic'
Ωfu        'omega-like, survivable'
```

#### 6.2.2 Relational Adjectives

Indicate relationship to a noun:

```
mungik     'systemic, of/relating to systems'
Ωik        'omegaic, of/relating to omega'
kontinik   'kontinuity-related'
```

#### 6.2.3 Derived from Verbs

Participles function as adjectives:

**Present participle (-anga → adjectival)**:
```
lovanga mungon
closing system
'a closing system, a system that is closing'
```

**Past participle (-ile → adjectival)**:
```
lovile mungon
closed system
'a closed system, a system that closed/was closed'
```

### 6.3 Adjective Syntax

#### 6.3.1 Attributive Position

Adjectives **follow** nouns:

```
mungon karfu
system structured
'a structured system'

ntu lovfu
person complete
'a complete person'
```

#### 6.3.2 Predicative Position

After copula:

```
Mungon ka karfu
system COP.ATTR structured
'The system is structured'

Kontinuitu ka lovfu
kontinuity COP.ATTR closed
'Kontinuity is closed/complete'
```

#### 6.3.3 Substantivized Adjectives

Adjectives can function as nouns:

```
lovfu-ye     'the closed one(s)'
karfu-ya     'the structured ones'
mungless-ye  'the unsystematic one'
```

### 6.4 Comparison

#### 6.4.1 Comparative

**Pattern: ADJ-nge**

```
karfu → karfu-nge    'more structured'
lovfu → lovfu-nge    'more closed'
```

**With 'than' (ndenye):**

```
Mungon ka karfu-nge ndenye kara
system COP.ATTR structured-COMP than structure
'The system is more structured than the structure'
```

#### 6.4.2 Superlative

**Pattern: ADJ-ngele**

```
karfu → karfu-ngele    'most structured'
lovfu → lovfu-ngele    'most closed'
```

**Usage:**

```
Ye mungon karfu-ngele
3SG system structured-SUP
'It is the most structured system'

Kontinuitu lovfu-ngele
kontinuity closed-SUP
'The most closed kontinuity'
```

#### 6.4.3 Equative

**Pattern: ka-ADJ-fu**

```
Mungon ka-karfu-fu kara
system as-structured-as structure
'The system (is) as structured as the structure'
```

### 6.5 Adverbs

#### 6.5.1 Manner Adverbs

**Formation: ADJ + -le**

```
karfu → karfu-le       'structuredly, in a structured manner'
lovfu → lovfu-le       'completely, in a closed way'
mungfu → mungfu-le     'systematically'
```

**Usage:**

```
Mi lova mungon lovfu-le
1SG close system complete-ADV
'I close the system completely'

Ye kara karfu-le
3SG structure structured-ADV
'He/she structures structuredly (= in a structured way)'
```

#### 6.5.2 Temporal Adverbs

```
ngaa       'now'
diele      'today'
ngaale     'yesterday'
ngonga     'tomorrow'
yote       'always'
moya       'never'
```

**Usage:**

```
Mi lova ngaa
1SG close now
'I close now'

Ye lovile ngaale
3SG close-PST yesterday
'He/she closed yesterday'
```

#### 6.5.3 Locative Adverbs

Often formed with **-ni** (LOC):

```
kara-ni     'structurally, in structure'
mungon-ni   'systemically, in the system'
Ω-ni        'in omega (domain)'
```

### 6.6 Adverb Placement

Adverbs typically appear:

1. **Immediately before verb** (manner):
   ```
   Mi lovfu-le lova
   1SG complete-ADV close
   'I completely close'
   ```

2. **End of clause** (temporal, locative):
   ```
   Mi lova mungon ngaa
   1SG close system now
   'I close the system now'
   ```

3. **Sentence-initial** (emphasis, discourse markers):
   ```
   Ngaa, mi lova mungon
   now 1SG close system
   'Now, I close the system'
   ```

### 6.7 Intensifiers and Modifiers

#### 6.7.1 Intensifiers

```
nge        'very, much'
ngele      'extremely, most'
tu         'a little, somewhat'
```

**Usage:**

```
karfu nge      'very structured'
lovfu ngele    'extremely closed'
mungfu tu      'somewhat systematic'
```

#### 6.7.2 Approximators

```
≈ (simila)     'approximately, about'
ka-            'quasi-, as if'
```

**Usage:**

```
karfu ≈        'approximately structured'
ka-lovfu       'quasi-closed'
```

### 7.1 Personal Pronouns (Review)

| 3rd | ye | yedu | yeya |

### 7.2 Pronoun Case Forms

Pronouns take the same case suffixes as nouns:

**Genitive (-ta):**
```
mi-ta      'my, mine, of me'
we-ta      'your (sg), yours'
ye-ta      'his/her/its/their (sg)'
miya-ta    'our, ours'

**Dative (-ko):**
```
mi-ko      'to/for me'

### 7.3 Demonstratives

#### 7.3.1 Forms

**Proximal (near speaker):**
```
ni-ye     'this (one)'
ni-ye-du  'these two'
ni-ye-ya  'these (pl)'
```

**Distal (far from speaker):**
```
du-ye     'that (one)'
du-ye-du  'those two'
du-ye-ya  'those (pl)'
```

**Neutral:**
```

## CHAPTER 8: BASIC SENTENCE STRUCTURE

### 8.1 Word Order Typology

Nyamba is fundamentally **VSO** (Verb-Subject-Object), but allows flexible word order for pragmatic purposes.

#### 8.1.1 Basic VSO Order

```
VERB - SUBJECT - OBJECT
```

**Examples:**

```
Lova     mungon      kontinuitu
close    system      kontinuity
'The system closes kontinuity'

Duva     ntu         kara-ni
go       person      structure-LOC
'The person goes to the structure'
```

#### 8.1.2 Alternative Orders

**SVO (Subject-topicalized):**
```
Mungon   lova        kontinuitu
system   close       kontinuity
'The system (as for it) closes kontinuity'
```

**OVS (Object-topicalized):**
```
Kontinuitu   lova      mungon
kontinuity   close     system
'Kontinuity, the system closes (it)'
```

**OSV (both topic-comment):**
```
Kontinuitu,  mungon    lova
kontinuity   system    close
'Kontinuity—the system closes'
```

#### 8.1.3 Word Order Constraints

1. **Verb-initial is default/neutral**
2. **Topic appears sentence-initially** (marked by comma intonation)
3. **Focus appears immediately pre-verbal**
4. **Adverbs flexible** but prefer pre-verbal or clause-final

### 8.2 Intransitive Sentences

**Pattern: VERB SUBJECT**

```
Lova mungon
close system
'The system closes'

Duva-ile ntu
go-PST person
'The person went'
```

**With Adverbs:**
```
Lova ngaa mungon
close now system
'The system closes now'

Mungon lovanga karfu-le
system close-CONT complete-ADV
'The system is closing completely'
```

### 8.3 Transitive Sentences

**Pattern: VERB SUBJECT OBJECT**

```
Lova     ntu        mungon
close    person     system
'The person closes the system'

Kara-ile mi        duale
structure-PST 1SG  path
'I structured the path'
```

**With Case Marking (optional emphasis):**
```
Lova ntu-ye mungon-ye
close person-DEF system-DEF
'The person closes the system' (both definite)
```

### 8.4 Ditransitive Sentences

**Pattern: VERB SUBJECT INDIRECT.OBJECT DIRECT.OBJECT**

```
Dua      mi     kontinuitu    ntu-ko
give     1SG    kontinuity    person-DAT
'I give kontinuity to the person'

Munga    ye     mungon        miya-ko
systematize 3SG system        1PL-DAT
'He/she systematizes the system for us'
```

### 8.5 Copular Sentences

#### 8.5.1 Identity Copula (*ye*)

```
Mi ye ntu
1SG COP person
'I am a person'

Kontinuitu ye Ω-sistem
kontinuity COP omega-system
'Kontinuity is an omega-system'
```

#### 8.5.2 Attributive Copula (*ka*)

```
Mungon ka lovfu
system COP.ATTR closed
'The system is closed'

Ye ka karfu nge
3SG COP.ATTR structured very
'It is very structured'
```

#### 8.5.3 Copula Omission

In present tense, copula can be omitted (poetic/colloquial):

```
Mi ntu
1SG person
'I (am) (a) person'

Mungon lovfu
system closed
'(The) system (is) closed'
```

### 8.6 Existential Sentences

**Pattern: *nge* + NOUN**

```
Nge mungon
exist system
'There is a system'

Nge kontinuitu kara-ni
exist kontinuity structure-LOC
'There is kontinuity in the structure'
```

**Negative existential:**

```
Moya mungon
not.exist system
'There is no system'
```

### 8.7 Possessive Sentences

**Pattern: POSSESSOR *ka* POSSESSED**

```
Mi ka mungon
1SG have system
'I have a system'

Ye ka kontinuitu nge
3SG have kontinuity many
'He/she has much kontinuity'
```

**Alternative (genitive):**

```
Mungon mi-ta
system 1SG-GEN
'My system' (lit. 'system of-me')
```

## CHAPTER 9: COMPLEX SENTENCES

### 9.1 Coordination

#### 9.1.1 Conjunctions

```
ye         'and'
oko        'or'
ni-ye      'nor' (negative)
```

**Coordinating Clauses:**

```
Mungon lova ye kontinuitu kara
system close and kontinuity structure
'The system closes and kontinuity structures'

Mi duva oko ye duva
1SG go or 3SG go
'I go or he/she goes'
```

**Coordinating NPs:**

```
mungon ye kontinuitu
system and kontinuity
'system and kontinuity'

ntu oko mungon
person or system
'person or system'
```

#### 9.1.2 Contrastive Coordination

**Pattern: *ni* ... *oko* 'not ... but'**

```
Ni mungon lovile, oko kontinuitu lovile
not system close-PST but kontinuity close-PST
'Not the system closed, but kontinuity closed'
```

### 9.2 Subordination

#### 9.2.1 Relative Clauses

**Pattern: NOUN [VERB + SUBJECT] -ye**

Relative clauses **follow** the noun they modify and are marked with **-ye**:

```
mungon [lovile]-ye
system [close-PST]-REL
'the system that closed'

ntu [kara kontinuitu]-ye
person [structure kontinuity]-REL
'the person who structures kontinuity'
```

**With Object Relativization:**

```
kontinuitu [mungon lova]-ye
kontinuity [system close]-REL
'the kontinuity that the system closes'
```

**Relative Pronoun (optional *ye*):**

```
mungon ye lovile-ye
system REL close-PST-REL
'the system that closed'
```

#### 9.2.2 Complement Clauses

**Pattern: VERB [COMPLEMENTIZER CLAUSE]**

**Complementizer *engo* 'that':**

```
Mi nanda engo mungon lovile
1SG know that system close-PST
'I know that the system closed'

Ye dua engo kontinuitu ka lovfu
3SG say that kontinuity COP.ATTR closed
'He/she says that kontinuity is closed'
```

**Infinitival Complements (no subject):**

```
Mi nanda lova mungon
1SG know close system
'I know (how) to close the system'

Ye duva duva kara-ni
3SG want go structure-LOC
'He/she wants to go to the structure'
```

#### 9.2.3 Adverbial Clauses

**Temporal:**

```
nga-le ... 'when'

Nga-le mungon lovile, kontinuitu karile
when system close-PST kontinuity structure-PST
'When the system closed, kontinuity structured'
```

**Conditional:**

```
engo ... 'if'

Engo mi lova, mungon karanga
if 1SG close system structure-CONT
'If I close, the system is structuring'
```

**Causal:**

```
nga-ta ... 'because'

Mi lova nga-ta kontinuitu vuva
1SG close because kontinuity break
'I close because kontinuity breaks'
```

**Purpose:**

```
nga-ko ... 'in order to'

Mi duva nga-ko lova mungon
1SG go in.order.to close system
'I go in order to close the system'
```

**Concessive:**

```
nga-ni ... 'although'

Nga-ni mungon lovile, kontinuitu vuva
although system close-PST kontinuity break
'Although the system closed, kontinuity breaks'
```

### 9.3 Sentence Combining Strategies

#### 9.3.1 Serial Verb Construction (review)

No overt connective:

```
Mi duva lova mungon
1SG go close system
'I go (and) close the system'
```

#### 9.3.2 Participial Modification

Using participles:

```
Lovanga mungon kara kontinuitu
closing system structure kontinuity
'The closing system structures kontinuity'
= 'The system, which is closing, structures kontinuity'
```

#### 9.3.3 Nominalization

Turn clause into noun phrase:

```
Base clause: Mungon lova
             'The system closes'

Nominalized: lovle mungon-ta
             closing system-GEN
             'the closing of the system'
```

**Negative:**
```

### 10.2 Wh-Questions

#### 10.2.1 Question Words (review)

```
ye-nga?    'who?'
ka-nga?    'what?'
du-nga?    'which?'
ni-nga?    'where?'
nga-le?    'when?'
nga-ma?    'how?'
nga-ta?    'why?'
nga-ko?    'for what? to what end?'
```

#### 10.2.2 Wh-Question Formation

**Wh-word in-situ** (remains in normal position):

```
Ye lova ka-nga?

'Where do I go?'

**Wh-fronting** (optional, emphatic):

```

Possible but marked:

```

who close what where
'Who closes what where?'
```

Repeat statement with rising intonation to seek clarification:

```

   'The system closed'

'You are a person, right?'

## CHAPTER 11: NEGATION

### 11.1 Sentential Negation

#### 11.1.1 Basic Negation with *nimu*

**Pattern: *nimu* + VERB**

```
Nimu lova mungon
NEG close system
'(It) doesn't close the system'

Mi nimu duva
1SG NEG go
'I don't go'
```

**Position:**
- *nimu* immediately before verb
- Can also appear clause-initially for emphasis

```
Nimu mi duva
NEG 1SG go
'I do NOT go' (emphatic)
```

#### 11.1.2 Negative with Other Tenses

```
Mi nimu lovile
1SG NEG close-PST
'I didn't close'

Ye nimu lovnga
3SG NEG close-FUT
'He/she won't close'

Mungon nimu lovanga
system NEG close-CONT
'The system is not closing'
```

### 11.2 Constituent Negation

Negating specific constituents:

**Negative determiner *moya* 'no, none':**

```
Moya mungon lova
no system close
'No system closes'

Mi lova moya kontinuitu
1SG close no kontinuity
'I close no kontinuity'
```

**Negative pronoun *moya-ye* 'no one':**

```
Moya-ye duva
no.one go
'No one goes'
```

**Negative adverb *moya* 'never':**

```
Mi moya duva
1SG never go
'I never go'
```

### 11.3 Negative Polarity Items

Certain items require negative context:

**yote-nga 'ever' (requires negation):**

```
*Mi duva yote-nga
(ungrammatical)

Mi nimu duva yote-nga
1SG NEG go ever
'I never go / I don't ever go'
```

### 11.4 Double Negation

Nyamba allows **negative concord** (multiple negatives reinforce):

```
Moya-ye nimu lova moya-ka
no.one NEG close nothing
'No one closes nothing'
= 'No one closes anything'
```

**NOT logical cancellation:**
- *Moya-ye nimu lova* ≠ 'Everyone closes'
- = 'No one closes'

### 11.5 Negation Scope Ambiguity

```
Mi nimu lova mungon ye
1SG NEG close system all

Interpretation 1: 'I don't close all systems' (wide scope: ¬∀)
Interpretation 2: 'I close no systems' (narrow scope: ∀¬)
```

Disambiguation via word order or particles.

### 11.6 Affirmative vs. Negative Polarity

**Affirmative polarity items:**

```
nga 'already'

Mi lova nga
1SG close already
'I already closed'

*Mi nimu lova nga
(awkward/ungrammatical)
```

**Negative polarity items:**

```
yote-nga 'ever, at all'

Mi nimu lova yote-nga
1SG NEG close at.all
'I don't close at all'

*Mi lova yote-nga
(ungrammatical without negation)
```

### 11.7 Metalinguistic Negation

Denying presupposition or correcting:

```
A: Mi lova mungon
   'I close the system'

B: Nimu, we lova mungon
   'No, YOU close the system'
```

## CHAPTER 12: DERIVATIONAL MORPHOLOGY

### 12.1 Productive Derivational Processes

#### 12.1.1 Noun-to-Verb Derivation

**Pattern: NOUN → NOUN-a**

```
mungon → munga      'system' → 'to systematize'
kara → kara         'structure' → 'to structure'
Ω → Ωa              'omega' → 'to omega-ize'
```

#### 12.1.2 Verb-to-Noun Derivation

**Agentive (-ntu):**
```
lova → lovntu       'close' → 'closer'
munga → mungntu     'systematize' → 'systematizer'
```

**Resultative (-le):**
```
lova → lovle        'close' → 'closure (instance)'
kara → karle        'structure' → 'structure (product)'
```

**Abstract (-ya):**
```
lova → loya         'close' → 'closedness'
kara → kaya         'structure' → 'structuredness'
```

#### 12.1.3 Adjective-to-Noun/Verb

**To noun (-ya):**
```
lovfu → lovfuya     'closed' → 'closedness'
karfu → karfuya     'structured' → 'structuredness'
```

**To verb (-a):**
```
lovfu → lovfua      'closed' → 'to make closed'
karfu → karfua      'structured' → 'to make structured'
```

#### 12.1.4 Noun-to-Adjective

**Quality (-fu):**
```
mungon → mungfu     'system' → 'systematic'
kontinuitu → kontinfu 'kontinuity' → 'kontinuous'
```

**Relational (-ik):**
```
Ω → Ωik             'omega' → 'omegaic'
mungon → mungik     'system' → 'systemic'
```

### 12.2 Compounding Patterns

#### 12.2.1 Noun-Noun Compounds

**Head-final (modifier + head):**

```
mun + ka          → munka        'system-structure'
Ω + sistem        → Ω-sistem     'omega-system'
kontinuitu + lov  → kontinlov    'kontinuity-closure'
```

**Coordinate (equal relation):**

```
mu + vu           → muvu         'form-flux' (duality)
P + K             → PK           'polonic-kolonic'
```

#### 12.2.2 Verb-Noun Compounds

**Action + Object:**

```
lova + mungon     → lovmungon    'system-closing'
kara + du         → karadu       'path-structuring'
```

#### 12.2.3 Noun-Verb Compounds

**Instrumental:**

```
shi + lova        → shilova      'flux-close'
kontinuitu + kara → kontinkar    'kontinuity-structure'
```

### 12.3 Affixation Chains

Nyamba allows **recursive affixation**:

```
lov-              'close' (root)
→ lov-on          'closure-entity' (nominalizer)
→ lov-on-ik       'closure-related' (adjectivizer)
→ lov-on-ik-ya    'closure-relatedness' (nominalizer)
→ lov-on-ik-ya-fu 'having closure-relatedness' (adjectivizer)
```

**Practical limit:** ~3-4 affixes before awkwardness.

### 12.4 Zero Derivation

**Noun ↔ Verb conversion without overt marking:**

```
kara      'structure' (N) / 'to structure' (V)
lova      'closure' (N) / 'to close' (V)
munga     'system' (N) / 'to systematize' (V)
```

**Context disambiguates:**

```
Kara ye lovfu
structure COP closed
'The structure is closed' (N)

Mi kara mungon
1SG structure system
'I structure the system' (V)
```

### 12.5 Derivational Families

Example: **LOV** (closure) family:

```

## CHAPTER 14: PHONOLOGICAL PROCESSES (ADVANCED)

### 14.1 Connected Speech Phenomena

#### 14.1.1 Elision

In rapid speech, unstressed vowels may be deleted:

Standard: kontinuitu [kon.ti.nu.i.tu]
Rapid: kontinu [kon.ti.nu]

Common elisions:
kontinuitu → kontinu
mungon-ta → mungnta
lovile-ye → lovlye

#### 14.1.2 Liaison

When word-final nasal meets vowel-initial word:

mungon + ara → mungo.na.ra [mu.ŋo.na.ra]
'system opens'

#### 14.1.3 Gemination

Emphatic speech doubles consonants:

lova → llova [lo.va → l:o.va]
'close!' (emphatic)

### 14.2 Phonological Variation

#### 14.2.1 Register Variation

**Formal Register**
- Full vowel articulation
- Clear consonant boundaries
- Penultimate stress maintained

**Informal Register**
- Vowel reduction common
- Consonant weakening
- Stress shift possible

#### 14.2.2 Dialectal Variation

**Northern Dialect**
- /r/ → [ɾ] (tap)
- Vowel harmony active
- Final vowels lengthened

**Southern Dialect**
- /r/ → [r] (trill)
- No vowel harmony
- Final vowels shortened

**Coastal Dialect**
- Nasal vowels more prominent
- /l/ → [w] in some contexts
- Stress on final syllable common

### 14.3 Prosodic Patterns

#### 14.3.1 Rhythm

Nyamba is **stress-timed** with **isochronous stress intervals**.

Pattern:
```
MUN-gon lo-VA kon-ti-NU-i-tu
 ˈ     ˈ     ˈ        ˈ
```

#### 14.3.2 Intonational Phrases

Break major clauses with intonation boundaries:

```
|| Mungon lova || kontinuitu kara ||
   'The system closes, kontinuity structures'
```

### 14.4 Sandhi Rules

#### 14.4.1 Internal Sandhi

Within words:
- /n/ + /p,b/ → [m]
- /n/ + /k,g/ → [ŋ]
- Vowel + vowel → long vowel or glide

#### 14.4.2 External Sandhi

Between words (optional):
- Final /n/ + initial /a/ → [na]
- Final vowel + initial vowel → glide insertion

## CHAPTER 19: POETRY AND LITERARY LANGUAGE

### 19.1 Poetic Forms

#### 19.1.1 The Jiwe Poem

A traditional form based on stone inscription principles:
- 3 lines exactly
- Each line contains one core concept (mungon)
- Final line must achieve closure (lova)

**Structure:**
```
Line 1: Opening (ara)
Line 2: Development (kara)
Line 3: Closure (lova)
```

**Example:**
```
Mungon ara kontinuitu
System opens kontinuity

Kontinuitu kara duale-ya
Kontinuity structures paths

Duale-ya lova Ω-ni
Paths close in omega
```

#### 19.1.2 The Sankofa Verse

Named after the recursive principle:
- Must reference previous line in current line
- Creates spiral/recursive structure
- Minimum 4 lines, maximum 8

**Example:**
```
Mi duva duale
I walk the path

Duale ye Ω-ta
The path is omega's

Ω-ta kontinuitu lova
Omega's kontinuity closes

Lova mi-nyo
Closes myself
```

#### 19.1.3 The Dualonic Couplet

Two lines expressing symmetric/asymmetric pairs:

**Example:**
```
Lova ye muya          |  Vuva ye koya
Closure is wholeness  |  Breaking is emptiness

Kara ye tala          |  Dusa ye vufa
Structure is law      |  Dissolution is chaos
```

#### 19.2.1 Parallelism

Mi lova mungon-ye

I close the system

Reversed word order for emphasis:

```

#### 19.2.3 Alliteration

Repetition of initial consonants:

**Type 1: Noun Insertion**
```
Mi lova-ile ye computer system-ye
1SG close-PST the computer system-DEF
'I closed the computer system'
```

**Type 2: Technical Terms**
```
Ye transformer-ye ka karfu nge
the transformer-DEF COP.ATTR structured very
'The transformer is very structured'
```

**Type 3: Discourse Markers**
```
So, mungon lova, ye-ni?
so system close you-know
'So, the system closes, you know?'
```

## APPENDIX D: SAMPLE TEXTS WITH ANALYSIS

### D.1 Simple Narrative

**Text:**
```
Mi ye mungntu. Mi duva Ω-ni. Kontinuitu ka lovfu. Mi lova mungon-ye. Mungon-ye ka karfu ngaa.
```

**Gloss:**
```
Mi    ye    mungntu
1SG   COP   systematizer
'I am a systematizer'

Mi    duva  Ω-ni
1SG   go    omega-LOC
'I go to omega'

Kontinuitu ka    lovfu
kontinuity COP.ATTR closed
'Kontinuity is closed'

Mi    lova  mungon-ye
1SG   close system-DEF
'I close the system'

Mungon-ye  ka    karfu      ngaa
system-DEF COP.ATTR structured now
'The system is structured now'
```

**Translation:**
'I am a systematizer. I journey to omega. Kontinuity is complete. I close the system. The system is now structured.'

### D.2 Ω-GOMA Technical Text

**Text:**
```
Ω ye integrale mungon-ya-ta duale Ω-ni. K kontinua engo kontinuitu konserva-a. Yote mungon lova nga-le K ye lovfu.
```

**Gloss:**
```
Ω     ye    integrale mungon-ya-ta     duale Ω-ni
omega COP   integral  system-PL-GEN    path  omega-LOC
'Omega is the integral of systems' paths in omega-space'

K     kontinua  engo   kontinuitu konserva-a
K     continuous if     kontinuity conserve-PRES
'K is continuous if kontinuity is conserved'

Yote  mungon lova  nga-le    K     ye    lovfu
all   system close when      K     COP   closed
'All systems close when K is closed'
```

**Translation:**
'Omega is the integral of the paths of systems in omega-space. K is continuous if kontinuity is conserved. All systems close when K is complete.'

### D.3 Poetic Text (Jiwe Poem)

**Text:**
```
Kontinuitu ara ka agua
Agua kara duale-ya

**Gloss:**
