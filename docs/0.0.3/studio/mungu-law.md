### Formal Statement

```
∀X : Exists(X) ⇒ ∃C such that Δ_C(X)
```

### Interpretation

Nothing exists unless it is distinguishable from non-itself within some context.

* No distinction → no existence
* Distinction is necessary, not sufficient, for persistence

## KORA LAW II — Closure

### Formal Statement

```
∀X : Persists(X) ⇒ K_C(X)
```
