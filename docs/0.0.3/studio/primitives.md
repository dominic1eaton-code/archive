### **Dualons & Trilons**

* **Dualon**: stable pair (A ↔ not-A)
* **Monon**: the unified whole
* **Trilon**: monon–dualon structure

* impi    : entity / component / thing
* tathri  : attribute / property
* indi    : type / class
* wekhi   : network / relation
* mali    : qualisense (quality, value, sense)
* ramani  : morphism / interaction / mapping
* sibi    : change between states
* sibon   : state itself


Mali: The fundamental general abstraction of Qualisense (characteristics, qualities, features, traits, aspects, properties). This term represents the core primitive for describing any type of attribute or quality of a component within the system.

Ramani: The fundamental general abstraction of Transmorphism (changes, transformations, interactions, morphisms). This term captures any form of dynamic transformation or interaction between components in the system.

Impi: The fundamental general abstraction of Elemona (components, elements, entities). This term represents the abstract concept of any component or element within the system, regardless of its specific role or behavior.

Tathri: The general abstraction of "type" in Mungu Theory. This term represents the concept of types that classify, define, or categorize components, behaviors, interactions, or any other entities in the system. It is the abstraction for classification within the broader framework.
