With capital vehicles active and governance fully delegated, the model transitions into **production activation**.

Within factories, **Studio entities** are created as **temporary execution vehicles** ‚Äî e.g., an Incubator Studio to spin up new products, a Renovation Studio for operational turnarounds, a Research Studio for new platform prototypes. As production entities solidify, **PortfolioCos** emerge to group and manage related ventures. Each PortfolioCo holds IP (via IPCo), manages incentive unit allocation (via Portfolio IncentiveCo), and serves as the governance umbrella for its constituent ventures. OVCS continues compliance monitoring; Commons councils continue to evolve standards based on feedback and network effects.
