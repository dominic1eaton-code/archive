# kogi platform
* the portfolio management platform for the independent worker


## entreprenuer operating system platform
- centralized platform app viewer: kogi-portal
- create portfolio of ideas with kogi-portfolio
    - portfolio ideas lifecycle management system
    - manage and administer job,work,career,project,creative,invesment,custom portfolios
- market,buy,sell,trade idea portfolios in the kogi-marketplace
- share,contribute,find support of portfolios in the kogi-community
- resource and fund portfolios with kogi-pay
- manage portfolio project, work, time, and resources with kogi-gig
- manage all portfolio related communications with kogi-chat
- extend platform functionality with kogi-dev;kogi-api
- base platform management, administration, distribution, configuration, deployment with kogi-host
